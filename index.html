Latitude vs. X

This project analyzes how weather changes as latitude varies! The analysis was created by making OpenWeatherMap API calls and gathering weather information on 500+ cities. The information was generated by CityPy and then exported to a csv so it could be be into an HTML table.

Matplotlib was then used to create the plots seen throughout the website. These plots display Max Temperature, Humidity, Wind Speed, and Cloud Coverage as latitude varies.
